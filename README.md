# Users CRUD Application #

CRUD Operations based on User's model

## Concept ##
- Basic Crud Operations

## Version 1 specs ##

### Technical Stack ###
* Data storage: in memory (array) 
* FrontEnd: React JS 
* State management: Redux
* Built using: Vite
* Architecture: To be defined 

### Available Functions ###
* Add a new User
* Update User data
* Delete an existing user
